# EVO
---
## Mô tả 
Đây là project luyện tập bài tập cơ bản về CRUD có liên quan tới HashSet
## Yêu cầu 
* Java [Phiên bản 17]
* Sử dụng Maven
* IDEA: Intellij
## Tải & Cài đặt 
1. Clone repository
```
https://github.com/snuppi19/EVO.git
```
3. Tải dependencies
 * backend
```
mvn install
```
5. Cấu hình kết nối database ở application.properties
6. Chạy backend
```
mvn spring-boot:run
```
7. chạy frontend
```
ng serve
```
